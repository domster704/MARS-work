Формирование шапки заказа

Итак. С чего начинается формирование заказа? Конечно же с заполнения его шапки.

Для заполнения шапки заказа открываем главное меню программы, нажав в левом верхнем углу значок в виде трех горизонтальных палочек.

В открывшемся главном меню программы находим пункт меню Шапка заказа и нажимаем на него. После чего нашему вниманию откроется раздел в котором мы можем указать реквизиты шапки нашего нового заказа.

В данном разделе мы видим несколько полей для заполнения, в правом верхнем углу две кнопки и в нижнем правом углу две кнопки. Разберем для чего нужно каждое поле и кнопка.

    Список полей для заполнения:

  • Торговый представитель
  • Поиск контрагента
  • Контрагент
  • Адрес доставки
  • Дата доставки
  • Время доставки
  • Комментарий
  • Забрать деньги у клиента
  • Забрать у клиента товар
  • Тип возврата

    • Список кнопок:

  • Сохранить заказ
  • Просмотр заказа
  • Дебиторка
  • Х
  • Очистить шапку заказа
  • Сохранить шапку заказа

Разберем по-порядку все поля и кнопки.

* Поле Торговый представитель - данное поле отображается в виде выпадающего списка, который содержит в себе всех торговых представителей компании. Список сортируется по имени торгового представителя.

Поле Поиск контрагента - данное поле отображается в виде поля для ввода текста и предназначено для поиска нужного контрагента.

Как пользоваться поиском контрагентов? На самом деле очень просто. Просто нажимаем на текстовое поле, и начинаем вводить наименование контрагента. По мере того как вы будете вводить наименование контрагента, список контрагентов в выпадающем списке будет сокращаться согласно критериям поиска. На рисунке ниже приведен пример. Контрагенты отфильтрованы по начальному слову сотр.

* Поле Контрагент - данное поле отображается в виде выпадающего писка, который содержит в себе контрагентов компании. Список контрагентов не простой, а золотой а детальный. При раскрывании списка контрагентов можно сразу обратить внимание на то, что некоторые контрагенты выделены другим цветом и через дробь у них написан статус.

Каждый контрагент может иметь четыре статуса и соответствующий цвет:

  • Обычный контрагент - отображается черным цветом (иными словами он не должник, не динамщик, на нем не стоит запрет отгрузок)
  • Должник - отображается сиреневым цветом и после наименования контрагента через дробь написано, что ДОЛЖНИК
  • Динамщик - отображается оранжевым цветом и после наименования контрагента через дробь написано, что ДИНАМЩИК
  • ЗАПРЕТ ОТГРУЗОК - отображается красным цветом и после наименования контрагента через дробь написано, что у него ЗАПРЕТ ОТГРУЗОК

Это сделано для того, чтобы на стадии формирования шапки заказа, каждый торговый представитель мог видеть статус контрагента, для которого создается заказ и в случае чего задуматься, а стоит ли?

* Поле Адрес доставки - данное поле отображается в виде выпадающего писка. По-умолчанию отображается пустым, но после выбора контргаента в нем будет отображен список всех адресов, по которым можно доставить товар контрагенту.

* Поле Дата доставки - данное поле отображается в виде поля для ввода. При нажатии на данное поле выводится календарь, в котором можно выбрать дату, когда товар будет доставлен контрагенту.

Поле Время доставки - данное поле отображается в виде поля для ввода. При нажатии на данное поле выводится временной календарь, в котором можно выбрать время, когда необходимо доставить товар контрагенту.

Поле Комментарий - данное поле отображается в виде поля для ввода. В данное поле вводится обычная текстовая информация, которую торговый представитель посчитает необходимой указать.

Поле Забрать деньги у клиента - данное поле отображается в виде поля "галочки". Если "галочка" установлена, то тем самым вы сообщаете офису о том, что у данного контрагента по тем или иным причинам необходимо забрать деньги.

Поле Забрать у клиента товар - данное поле отображается в виде поля "галочки". Если "галочка" установлена, то автоматически разблокируется поле "Тип возврата", описание которого приведено ниже. Эти два поля связаны между собой и работают в связке.

Поле Тип возврата - данное поле отображается в виде выпадающего списка и содержит список причин, по которым у клиента необходимо забрать товар.

Кнопка Сохранить шапку заказа - предназначена для того чтобы сохранить все введеные реквизиты шапки заказа.

Кнопка Очистить шапку заказа - предназначена для того чтобы сбросить все поля шапки заказа, за исключением поля Торговый представитель.

Кнопка Дебиторка - если в поле Контрагент указан контрагент, для которого формируется заказ, то откроется раздел дебиторской задолженности для данного контрагента.

Кнопка Х - банально очищает поле "Время доставки", так как оно не обязательное и может быть пустым.

ПОЛЯ, ОТМЕЧЕННЫЕ ЗНАКОМ * ОБЯЗАТЕЛЬНЫ ДЛЯ ЗАПОЛНЕНИЯ. ЕСЛИ ДАННЫЕ ПОЛЯ НЕ ЗАПОЛНЕНЫ, ТО ВЫ НЕ СМОЖЕТЕ СОХРАНИТЬ ШАПКУ ЗАКАЗА.

Итак, что обозначает каждое поле в разделе Шапка заказа мы разобрали. После того как все обязательные поля заполнены, нажимаем на кнопку СОХРАНИТЬ ШАПКУ ЗАКАЗА, программа сообщает нам о том, что она успешно сохранена. Теперь мы можем перейти непосредственно к формированию самого заказа.
